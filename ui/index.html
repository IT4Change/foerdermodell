Statische Demo-Ansichten des Förderprozesses, alle auf demselben Beispieldatensatz. Kein Login, keine echten Daten — zum Zeigen gedacht.

So funktioniert das Initiativkonto

Ein Gemeinschaftskonto. Dreimal im Jahr ein Stichtag: Mitglieder reichen Initiativen ein. Der Kontostand wird zu gleichen Teilen auf alle stimmberechtigten Mitglieder aufgeteilt — jedes verteilt seinen Anteil frei auf die Initiativen. Nicht vergebene Anteile bleiben auf dem Konto.
